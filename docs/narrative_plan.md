# Narrative Plan

## Audience

Преподаватель и студенты, оценивающие понимание Vue.js через сочетание теории и интерактивной практики.

## Objective

Показать, что работа охватывает ключевые темы Vue.js: современный запуск проекта, Single File Components, реактивность, жизненный цикл, передачу props и связь вопросов с реальным кодом.

## Narrative Arc

1. Старт миссии: почему Vue.js и почему сегодня чаще выбирают Vite.
2. Архитектура: как строится приложение из компонентов.
3. Динамика: как работает реактивность.
4. Финальная интеграция: как данные передаются между компонентами.
5. Защита: как Kahoot-квест и demo-проект дополняют друг друга.

## Slide List

1. Титульный слайд.
2. Цель и формат работы.
3. Сюжет квеста.
4. Структура demo-проекта.
5. Vue concepts in code.
6. Kahoot question design.
7. Итоги и ссылки.

## Visual System

- Светлая академическая палитра: песочный, тёмно-синий, бирюзовый, янтарный.
- Заголовки в геометрическом гротеске, основной текст в neutral modern sans.
- Карточная композиция с мягкими градиентами и маршрутной графикой.

## Asset Needs

- Скриншот интерфейса demo-проекта.
- Иконографика маршрута из 4 этапов.

## Editability Plan

Все тексты, заголовки и подписи должны быть сохранены как редактируемые объекты PowerPoint.
