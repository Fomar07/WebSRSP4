Как тема применяется на нескольких страницах

Эта страница показывает, как один и тот же набор CSS-переменных и один JavaScript-файл позволяют поддерживать единое поведение темы во всем проекте.

1. Общий CSS

Все страницы используют единый набор переменных для фона, текста, акцентов и карточек.

2. Общий JavaScript

Один файл управляет переключением темы, чтением localStorage и восстановлением состояния.

3. Общий ключ хранения

Во всех страницах используется одно значение preferred-theme.

Почему это удобно

Для небольшого проекта

  • не нужно дублировать код на каждой странице
  • достаточно подключить один JS-файл
  • тема выглядит одинаково во всех разделах

Для расширения

  • можно вынести тему в отдельный модуль
  • можно добавить новые варианты оформления
  • можно развивать проект без переписывания базы
